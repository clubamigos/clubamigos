📜 Tournament Rules

⚽ Match Prediction

Predict before matches are locked. All daily matches will be locked at 20:00 (8:00 PM). Once locked, predictions cannot be edited.

  • For knockout matches that go to penalties, the final score including penalty goals will be considered. Example: If the match ends 1 - 1 and penalties finish 4 - 2, the final result will be counted as 5 - 3 for prediction scoring.
  • Exact score = 5 PTS
  • One correct team score = 2 PTS
  • Wrong prediction = 0 PTS

🏆 Group Stage

Group Winners, Runners-up and Best 3rd must be selected before the first match begins. After the first match starts, group predictions will be locked.

  • Correct Winner = 3 PTS
  • Correct Runner-up = 3 PTS
  • Best 3rd = 3 PTS

🔥 Knockout Stage

Knockout stage predictions must be completed before the first knockout match begins. After that, all predictions will be locked.

  • R32 Winner = 5 PTS
  • R16 Winner = 7 PTS
  • Quarter Final = 10 PTS
  • Semi Final = 15 PTS
  • 3rd Place Match = 10 PTS
  • Final Winner = 20 PTS

📌 Important

  • Always complete predictions before lock time
  • Daily match lock time is 20:00 (8:00 PM)
  • Group stage locks before first match
  • Knockout stage locks before first knockout match
  • Leaderboard updates automatically
  • Save predictions before leaving page
